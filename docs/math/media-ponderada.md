- Também conhecida como `média aritmética ponderada`, ela é aplicada quando há pesos nos valores dos conjuntos.

- Fórmula:

$$
x = \frac {x_1 * p_1 + x_2 * p_2 + x_3 * p_3 + ... x_n * p_n} {p_1 + p_2 + p_3 + ... + p_n}
$$
